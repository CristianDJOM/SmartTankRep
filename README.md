# 💧 SmartTank

## 📖 Project Description

SmartTank is an Internet of Things (IoT) solution designed to monitor water tank levels in real time. The system combines an ESP32 microcontroller, ultrasonic sensing technology, and a mobile application developed with React Native to provide remote access to water level information.

The ultrasonic sensor continuously measures the distance between the sensor and the water surface, allowing the system to calculate the tank filling percentage. The collected data is transmitted through the ESP32 and displayed in a user-friendly mobile application, enabling users to monitor tank status from anywhere.

SmartTank helps prevent water shortages, improves resource management, and provides a practical example of IoT integration between hardware and software systems.
